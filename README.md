✈️ British Airways Reviews Dashboard

Dashboard visualisasi interaktif menggunakan Tableau untuk menganalisis ulasan pelanggan British Airways. Dashboard ini menyediakan berbagai filter dinamis sehingga pengguna dapat mengeksplorasi data secara fleksibel berdasarkan atribut ulasan, waktu, negara, dan jenis pesawat.

✨ Fitur
- 📈 Line Chart untuk menampilkan rata-rata skor ulasan (Overall Rating, Cabin Staff Service, Entertainment, dan lainnya) berdasarkan bulan.
- 🌍 Geographical Chart untuk memvisualisasikan rata-rata skor ulasan berdasarkan negara.
- ✈️ Bar Chart untuk membandingkan rata-rata skor ulasan berdasarkan jenis pesawat.
- 🎛️ Filter interaktif yang memungkinkan eksplorasi data secara dinamis.

🛠️ Teknologi
- Tableau
- British Airways Reviews Dataset
